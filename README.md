# Controle de Lucro IPTV - Canais Flex

Sistema PWA simples para controlar vendas, custos e lucro de acessos.

## Arquivos principais

- `index.html`: sistema completo
- `manifest.webmanifest`: configuração PWA
- `sw.js`: service worker/offline
- `icons/`: ícones do app
- `vercel.json`: configuração para deploy na Vercel

## Como subir na Vercel

1. Crie um repositório no GitHub.
2. Envie todos estes arquivos para o repositório.
3. Entre na Vercel.
4. Clique em **Add New > Project**.
5. Importe o repositório do GitHub.
6. Em framework, escolha **Other**.
7. Deixe build command vazio.
8. Deixe output directory vazio ou `.`.
9. Clique em **Deploy**.

Depois do deploy, abra o link HTTPS pelo Chrome no celular e toque em **Instalar app**.
